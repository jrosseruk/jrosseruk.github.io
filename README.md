## Tech Stack

- Vanilla HTML, CSS, JavaScript
- Markdown content
- GitHub Pages hosting

## Development

```bash
# Serve locally
python -m http.server 8000 -d docs/
```

## Structure

- `docs/index.html` - Main page
- `docs/*.md` - Content (about, publications, resume)
- `docs/styles.css` - Styling
- `docs/script.js` - Functionality
